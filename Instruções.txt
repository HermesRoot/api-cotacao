# Instruções de Uso

1. Navegue até a pasta do projeto:
   - Abra o terminal (ou prompt de comando) e execute:
     cd /caminho/para/o/projeto
   - Substitua "/caminho/para/o/projeto" pelo diretório onde o repositório foi clonado.

2. Execute o arquivo Python:
   - Digite o seguinte comando e pressione Enter:
     python cotacao_api.py
   - Certifique-se de que as dependências estão instaladas antes de rodar o script.

3. Verifique se a API está online:
   - Abra seu navegador e acesse:
     http://127.0.0.1:5000/cotacao
